## Demo
[here](http://3.15.0.35:3000/)

## Env variables
- `SECRET_KEY`: Random string to make encryption secure
- `SQLITE_DATABASE`: SQLite DB path

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
